GlimChat

Simple CSS to overlay your Glimesh Chat in OBS/XSplit. No signups, no selfhosting.

Installation

Create a new browser source in OBS/XSplit, and set the URL to https://glimesh.tv/CHANNELNAMEHERE/chat, and replace CHANNELNAMEHERE with your channel name. Then, go down to custom CSS, and replace the prefilled CSS with my CSS below. Then simply hit OK! 
  /* GlimChat CSS v1.1 by BurritoSOFTWARE */
body{background-color:rgba(0,0,0,0);margin:0 auto;overflow:hidden}.input-group{display:none}#chat-messages.chat-messages{background-image:none}::-webkit-scrollbar{width:0;height:0}#channel-header{display:none}.chat-mod-icon{display: none !important}.fas{display:none}
